# Security Policy

## Reporting

For v0.1 Beta, please report security concerns through GitHub Issues in the
public repository. Include enough detail to reproduce the issue, but do not post
live API keys, private URLs, personal memory contents, or other secrets.

If the repository later adds a private disclosure channel, this file should be
updated before the next public release.

## Scope

This repository contains the Hermes iOS Channel app and its lightweight channel
server. Hermes itself owns provider keys, model configuration, tools, and the
primary runtime. Security reports for Hermes core should be filed with the
Hermes project.
